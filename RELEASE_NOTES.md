es4all 三平台（armbian / rocknix / emuelec）aarch64 构建。
每个 zip 内含 emulationstation（主程序）+ resources + locale，解压后覆盖到设备对应目录即可，无需重新编译，重启 EmulationStation 生效。

## 1.1 主要更新

### 新增
- **软件更新（OTA）**：直接从 GitHub Releases 检查并安装新版，更新完可一键重启套用；提示里带版本日期时间。
- **音源输出**：平台设置里切换 HDMI / AV（3.5mm）/ 光纤。三个平台各走各的后端 —— EmuELEC 用 `setauddev` 并同步开关 HDMI 硬件路径（修「选了 AV，HDMI 仍同时出声」）、ROCKNIX 走 PipeWire、Armbian 直接写 ALSA 配置。机型与可用输出的对应表在 `resources/audio_outputs.cfg`，加新机型只需加一行、不必重编。
- **CPU 调速器**：性能 / 智能调度 / 按需 / 保守 / 省电，即时生效并在开机时还原。
- **RetroArch 日志**：改成 RetroArch 自己的四级（调试 / 信息 / 警告 / 错误）+ 关闭，预设关闭。
- **蓝牙**：补回「手动配对蓝牙设备」入口。

### 改进
- **启动画面**：近黑副标画在黑底上几乎看不见（上游原有的毛病），改为白底，三个平台一致。
- 预设系统音量与音乐音量改为 80（原本音乐音量的预设值超出滑杆范围）。
- 中英文之间的空白规则统一：选项名紧贴、说明保留空白。
- 移除重复选项：RetroArch 菜单样式、RA 边框（与「游戏设置」里的同一个键重复）。

### 修复
- **更新后新翻译显示英文**：locale 的挂载点从未被建立，导致挂载静默失败、一直读旧的翻译。
- **更新后字型缺失/乱码**：更新流程会在 ES 运行中删掉自己正在读的资源目录，改为先暂存、开机时才替换。
- **部分机型音源输出菜单不出现**：读取外部命令输出时会吃掉最后一个字元，机型名比对因此失败。
- **更新后第一次开机黑屏约 30 秒**（ROCKNIX）：更新时把系统已编译好的 locale 一起删掉，开机得重编一次。

### 移除（后端做不到，留着只会误导）
- **视频模式**：三个平台一律移除，后端保留，待验证手段完善后再接回。
- **外部挂载选项**：Armbian 与 ROCKNIX 移除入口 —— Armbian 缺整套挂载机制，ROCKNIX 侧外接碟因枚举时序问题挂不上。详见 README 的「待办（v1.2）」。
